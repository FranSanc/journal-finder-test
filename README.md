# Journal Finder Test

## About

This project is a Vite + React journal finder app. It helps match research abstracts to suitable Frontiers journal options.

## Getting Started

1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies: `npm install`
4. Start the local dev server: `npm run dev`

## Environment

The app uses Vite environment variables as needed. For local development, add any required `VITE_*` settings to `.env.local`.

## Build

Run `npm run build` to create a production bundle.
